# OrangeHRM QA Manual

![Status](https://img.shields.io/badge/status-concluído-brightgreen)

![QA](https://img.shields.io/badge/QA-Manual-blue)

![GitHub](https://img.shields.io/badge/Git-Versionado-orange)

Projeto completo de **Qualidade de Software (QA Manual)** desenvolvido sobre a aplicação **OrangeHRM Demo**, com foco no planejamento, execução e documentação de testes funcionais.

O projeto simula as atividades realizadas por um Analista de QA em um ambiente corporativo, abrangendo desde o planejamento dos testes até a consolidação dos resultados, utilizando boas práticas de documentação e versionamento com Git e GitHub.

---

## 📖 Sobre o Projeto

O objetivo deste projeto foi aplicar, na prática, conceitos de QA Manual utilizando a aplicação OrangeHRM Demo como sistema sob teste.

Durante o desenvolvimento foram produzidos os principais artefatos utilizados em projetos reais de Qualidade de Software, incluindo plano de testes, casos de teste, evidências de execução, matriz de cobertura e relatório de execução.

Todo o projeto foi organizado utilizando documentação estruturada, Git e GitHub, simulando um fluxo de trabalho profissional adotado por equipes de Qualidade de Software.

---

## 🎯 Objetivos

- Planejar os testes da aplicação.
- Elaborar casos de teste funcionais.
- Executar os cenários planejados.
- Registrar evidências de execução.
- Consolidar a cobertura dos testes.
- Documentar os resultados obtidos.
- Organizar o projeto utilizando Git e GitHub.
