# OrangeHRM QA Manual

Projeto de QA Manual desenvolvido para fins de portfólio, utilizando a aplicação OrangeHRM Demo.

## Objetivo

Validar funcionalidades críticas da aplicação através de testes manuais, documentação de casos de teste, evidências, relatório de bugs e matriz de cobertura.
